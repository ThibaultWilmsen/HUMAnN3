# HUMAnN3
HUMAnN3 is a tool designed to accurately and efficiently determine the abundance of microbial metabolic pathways and molecular functions from metagenomic or metatranscriptomic sequencing data. This tool will be explored during the internship

## Setup
To use HUMAnN3 a conda environment was created with al the necessary tools. A humman3.yml file was made availible in this github repository.
1. conda env create -f humann3_env.yml
2. conda activate humann3_env
3. Download databases (see below)
